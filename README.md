# Elysian Real Estate

A luxury real estate web application built with React + Vite.

## Tech Stack

- **React** with React Router DOM
- **Vite** (build tool)
- **Tailwind CSS** (custom theme: ivory, charcoal, gold)
- **Lucide React** (icons)
- **AOS** (scroll animations)
- **Swiper.js** (testimonial slider)
- **EmailJS** (contact form)
## Pages

| Route | Description |
|-------|-------------|
| `/` | Home — hero, properties, services, agents, testimonials |
| `/properties` | Browse & filter properties by type, city, country |
| `/agents` | Meet the team |
| `/about` | Company timeline & stats |
| `/contact` | Contact form + map |
| `/login` | User login |
| `/signup` | New account |

## Getting Started

```bash
npm install
npm run dev 
```

## Features

- Property filtering by type, city, and country
- Property details modal
- User authentication (localStorage-based)
- Contact form via EmailJS
- Responsive design
